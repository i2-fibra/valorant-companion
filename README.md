# VAL//OPS — Killjoy Companion

Companion app pessoal para Valorant, focado em setups da Killjoy.

Catálogo organizado por mapa → bomb (A/B/C) → tipo de utility (torreta, granada, alarme, combo, ult) → lado (defesa, ataque execução, ataque pós-plant).

## Acesso

Publicado via GitHub Pages: https://i2-fibra.github.io/valorant-companion/

App standalone — HTML único com CSS, JS e imagens embutidas. Abre no navegador durante a partida.
